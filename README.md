# Snake

Classic snake game with adjustable board size and progressive difficulty.

Built with React 19 + TypeScript using canvas rendering and [strict-redux](https://github.com/aszusz/zero-to-react/tree/strict-redux) - a minimal, type-safe Redux variant.

Created from [aszusz/zero-to-react](https://github.com/aszusz/zero-to-react) template (strict-redux branch).

## Features

- Arrow key controls
- Adjustable board dimensions (10-30)
- Level progression with increasing speed
- Score tracking

## Scripts

- `npm run dev` – dev server
- `npm run build` – production build
- `npm test` – run tests
- `npm run all` – format, lint, test
